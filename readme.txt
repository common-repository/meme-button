Aggiungi il meme button nel tuo sito

== Description ==

Aggiungi il meme button nel tuo sito velocemente.

Se non sei iscritto registrati anche tu su <a href="http://meemi.com/p/signup/enzolarosa">Meemi.com</a>.

Scegli dove inserire il Meme Button (pagine o articoli, o entrambi).

== Installation ==

Scarica l'archivio, ed estrai i file, e successivamente:

1. Carica la cartella memi-button nella cartella `/wp-content/plugins/` del tuo sito
1. Attiva il plugin della pagina 'Plugins' la trovi nel menu di WordPress
1. Configura il plugin dalla pagina Impostazioni
